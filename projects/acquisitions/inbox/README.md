# Inbox

Drop zone for listings Matt encounters outside the discovery task: broker email, referral, a link from a friend, a listing he spotted while on a marketplace directly.

Workflow:

1. Paste listing content or URL into a new file: `inbox/YYYY-MM-DD-[slug].md`
2. Next session, Claude processes inbox items: creates `deals/[slug]/listing.md` using the intake template, then moves the inbox file to `deals/[slug]/source.md`
3. Inbox stays empty between sessions

Items here are pre-triage. They have not been filtered against the thesis yet.
