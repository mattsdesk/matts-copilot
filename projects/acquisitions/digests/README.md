# Digests

Output of the discovery scheduled task. One file per run, named `YYYY-MM-DD.md`.

Format follows `../templates/daily-digest.md`.

Matt skims each digest and fills in the "Promotion Actions" checklist at the bottom. Promoted listings move to `../deals/[slug]/`.

Older digests are kept for pattern analysis (what sources are productive, what thesis fit looks like over time).
